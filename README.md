
Landing Page desenvolvida como projeto da disciplina de Programação Front-End e utilizada como meu portfólio pessoal.

## 📖 Sobre o projeto

Este projeto foi desenvolvido utilizando HTML5, CSS3 e JavaScript, com foco em boas práticas de desenvolvimento Front-End, estrutura semântica e design responsivo.

A página apresenta informações sobre mim, minhas habilidades, projetos e formas de contato.

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- Google Fonts

## ✨ Funcionalidades

- Layout responsivo
- Menu de navegação
- Seção Sobre Mim
- Habilidades
- Projetos
- Informações extras
- Rodapé com contato
- Animações suaves
